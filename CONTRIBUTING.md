# Contribution à Casier Chap 🍻

Merci de l'intérêt que vous portez à Casier Chap ! Nous accueillons avec plaisir les contributions pour améliorer le quotidien des gérants de maquis.

## 🚀 Comment contribuer ?

1.  **Signaler un bug** : Utilisez nos [templates d'issues](https://github.com/velesky/Casier-shap/issues/new/choose) pour signaler un problème.
2.  **Proposer une fonctionnalité** : Nous sommes ouverts à de nouvelles idées !
3.  **Soumettre une Pull Request** :
    *   Forkez le dépôt.
    *   Créez une branche pour votre fonctionnalité (`git checkout -b feature/ma-fonctionnalite`).
    *   Testez vos changements sur un émulateur.
    *   Assurez-vous que `flutter analyze` passe sans erreur.
    *   Ouvrez une Pull Request avec une description claire.

## 🎨 Règles de Design

Tout changement visuel doit respecter le **Design System** (Dark Mode, Orange #FF7A00, Glassmorphism). Consultez `Documentations/UI_UX_doc.md` pour plus de détails.

## 🌍 Langue

Le code est en anglais pour les termes techniques, mais les commentaires et la documentation destinés aux utilisateurs finaux ou aux contributeurs locaux doivent être en **français**.

Merci de participer à l'aventure ! 🍹
